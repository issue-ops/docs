## Sensitive information

<DoDontContainer stacked>
  <Do indented>
    <Caption>
      Use issue forms inputs that accept references to sensitive information in
      secure locations
    </Caption>
  </Do>
  <Dont indented>
    <Caption>Accept sensitive information directly in issues</Caption>
  </Dont>
</DoDontContainer>

## Validation

<DoDontContainer stacked>
  <Do indented>
    <Caption>
      Validate issue and comment text at every step in the IssueOps workflow
    </Caption>
  </Do>
  <Dont indented>
    <Caption>
      Rely on labels to determine if an issue has been validated or approved
    </Caption>
  </Dont>
</DoDontContainer>
